# Saída
Os principais métodos de saída são `echo` e `print`.

```php
<?php
$nome = "Ewerton";
$idade = 21;

echo 'Nome: '.$nome; // Output: Nome: Ewerton
echo 'Idade: '.$idade; // Output: Idade: 21
echo 'Nome: $nome'; // Output: Nome: $nome
echo 'Idade: $idade'; // Output: Idade: $idade

echo "Nome: ".$nome; // Output: Nome: Ewerton
echo "Idade: ".$idade; // Output: Idade: 21
echo "Nome: $nome"; // Output: Nome: Ewerton
echo "Idade: $idade"; // Output: Idade: 21
?>
```
